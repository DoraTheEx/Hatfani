פרוטוקול IP (Internet Protocol) הוא פרוטוקול רשת סטנדרטי המאפשר תקשורת בין מחשבים ברשת. הוא אחראי על ניתוב חבילות נתונים (המכונות "פקטים") ממקור ליעד דרך רשתות IP.

פקטת IP היא יחידת הנתונים הבסיסית של פרוטוקול IP. היא מכילה את המידע הדרוש לניתוב הפקט ליעדו, כולל:

כתובת IP מקור: כתובת ייחודית המזהה את המחשב ששלח את הפקט.
כתובת IP יעד: כתובת ייחודית המזהה את המחשב שאמור לקבל את הפקט.
סוג הפרוטוקול: מזהה את סוג הפרוטוקול שבו נעשה שימוש בשכבה העליונה (לדוגמה, TCP, UDP, ICMP).
נתונים: המידע בפועל שמועבר בין המחשבים.
גודל פקטת IP נע בין 64 ל-1500 בתים.

תהליך ניתוב פקט IP:

המחשב המקור מחשב את כתובת ה-IP של היעד.
המחשב המקור מחלק את הנתונים לפקטים.
לכל פקט מתווספת כותרת IP המכילה את כתובות המקור והיעד.
הפקטים מועברים דרך רשתות IP עד שהם מגיעים ליעדם.
המחשב היעד מרכיב את הפקטים חזרה למידע המקורי.
